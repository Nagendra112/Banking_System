# Banking System Project using C++
* A simple code in C++ on banking system Can perform basic tasks like :-

     Select one option below -
     
        1. Open an Account       
        2. Balance Enquiry       
        3. Deposit
        4. Withdrawal
        5. Close an Account      
        6. Show All Accounts     
        7. Quit 

> It stores previous data using file handling.
